# RSV 360 - Guest Web App

App web para hóspedes (busca, detalhes, reserva, pagamentos).

## Desenvolvimento

```bash
npm run dev
```

Servidor iniciará na porta **3000**.

## Estrutura

- `pages/` - Páginas do guest app (será preenchido na Fase 2)
- `components/` - Componentes específicos do guest (será preenchido na Fase 2)
- `lib/` - Bibliotecas e utilitários (será preenchido na Fase 2)
