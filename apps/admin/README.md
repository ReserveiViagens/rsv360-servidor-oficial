# RSV 360 - Admin Panel

Painel administrativo para hotéis/proprietários/operadores.

## Desenvolvimento

```bash
npm run dev
```

Servidor iniciará na porta **3001**.

## Estrutura

- `pages/` - Páginas do admin panel (será preenchido na Fase 2)
- `components/` - Componentes específicos do admin (será preenchido na Fase 2)
- `lib/` - Bibliotecas e utilitários (será preenchido na Fase 2)
