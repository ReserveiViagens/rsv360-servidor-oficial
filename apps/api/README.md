# RSV 360 - API Backend

APIs HTTP/GraphQL usadas por guest + admin + parceiros.

## Desenvolvimento

```bash
npm run dev
```

Servidor iniciará na porta **5000** (padrão) ou conforme variável de ambiente `PORT`.

## Estrutura

- `src/routes/` - Rotas da API (será preenchido na Fase 2)
- `src/services/` - Serviços de negócio (será preenchido na Fase 2)
- `src/middleware/` - Middlewares (será preenchido na Fase 2)
- `src/config/` - Configurações (será preenchido na Fase 2)

## Health Check

```bash
curl http://localhost:5000/health
```
